# Создание сайта визитки/портфолио 
## Для клонирования репозитория
```bash 
git clone https://github.com/LamberJacks/guap.git
```
### Минимальные требования к выполнению задания:
1. Размещение и доступность
Если используется Tilda, Umi.ru — предоставить ссылку на их хостинге.
Если сайт написан вручную — разместить на доступном в РФ хостинге (Timeweb, Reg.ru, Yandex Cloud).
Приложить QR-код для быстрого перехода.
2. Контент и уникальность
Текст: минимум 300 символов (не шаблонный, написанный самостоятельно или переработанный).
Изображения:
1 оригинальное фото (например, ваше или ваших работ, не стоковое).
1 адаптивная графика (например, SVG-иконка или оптимизированное изображение).
3. Адаптивность и доработка
Проверить корректность отображения на мобильных устройствах (через DevTools или реальные тесты).
4. Навигационное меню
Создать меню с 3-4 разделами (например: "Главная", "Обо мне", "Портфолио", "Контакты")
Реализовать плавную прокрутку к разделам при клике на пункты меню
Для конструкторов: использовать встроенный инструмент создания меню
Для ручной верстки: реализовать через теги `<nav>` и `<a href="#section-id">`
5. Галерея изображений
Добавить раздел с 3-5 тематическими изображениями
Организовать изображения в сетку (2-3 колонки)
Для конструкторов: использовать блок "Галерея"
Для ручной верстки: применить CSS Grid или Flexbox
6. Форма обратной связи
Добавить форму с полями: Имя, Email, Сообщение
Реализовать валидацию email (проверка на наличие @)
Для конструкторов: использовать готовый виджет формы
Для ручной верстки: теги ```<form>```, ```<input>```, `<textarea>`
7. Футер сайта
Создать подвал страницы с копирайтом и годом создания;
Добавить иконки соцсетей с ссылками (минимум 2);
Для конструкторов: использовать блок "Футер";
Для ручной верстки: тег `<footer>` с CSS-стилизацией;
Телефон и email должны быть кликабельными: (tel:+79991234567, mailto:example@mail.ru).
